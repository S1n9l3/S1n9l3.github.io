[java] java的http请求方法

java的http请求方法

java实现http请求的三种方式：

  1. HttpURLConnection，java标准类
  2. org.apache.commons.httpclient
  3. org.apache.http.client

还是更习惯用HttpURLConnection。

HttpURLConnection

HttpURLConnection是java标准类，使用时需要注意，传入的url必须带协议的，比如http://wxyy.one/，而不能wxyy.one。

如果需要跟随跳转的话，就需要得到重定向地址，再次发送请求。
