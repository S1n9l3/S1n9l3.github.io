codeql语法-变量

QL中的变量与代数或逻辑中的变量的使⽤⽅式相似。它们表示⼀组值，这些值通常受公式限制。
这与某些其他编程语⾔中的变量不同，在其他编程语⾔中，变量表示可能包含数据的内存位置。该数据还可以随着时间变化。例如，在QL中， n = n + 1 是⼀个等式公式，仅当 n 等于 n + 1 时才成⽴（因此，实际上它不适⽤于任何数值）。在Java中， n = n + 1 不是等式，⽽是通过添加 1 到当前的值 n ，并进⾏新的赋值来改变 n 代表的值。
